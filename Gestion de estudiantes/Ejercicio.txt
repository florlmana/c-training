Escribe un programa en C que permita gestionar información de estudiantes. El programa debe ser capaz de realizar las siguientes operaciones a través de un menú:

Agregar un estudiante: El usuario debe poder ingresar el nombre y la edad de un estudiante. La información del estudiante debe almacenarse en una estructura de datos dinámica.

Mostrar estudiantes: El programa debe mostrar la lista de estudiantes ingresados hasta el momento, incluyendo sus nombres y edades.

Eliminar un estudiante: El usuario debe poder eliminar un estudiante específico de la lista. Al eliminar un estudiante, la memoria asignada para ese estudiante debe ser liberada.

Salir del programa: El usuario debe tener la opción de salir del programa.

Utiliza punteros y la función malloc para gestionar la memoria de manera dinámica. Asegúrate de manejar correctamente la asignación y liberación de memoria para evitar fugas de memoria.

Además, implementa un menú que permita al usuario seleccionar las operaciones mencionadas. Cada vez que el usuario realice una operación, debe regresar al menú principal para seleccionar otra operación o salir del programa.
